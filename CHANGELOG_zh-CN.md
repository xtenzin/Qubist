# 变更日志

本文档记录项目的所有重要变更。

## [未发布]

### 计划中
- 向量相似度搜索功能
- 批量操作功能

## [0.1.0] - 2026-02-04

### 新增
- 初始版本发布
- 连接管理功能
- 集合管理功能（列表、查看、创建、设置）
- 点数据 CRUD 操作
- 分页功能
- 多语言支持（中文、英文）
- 响应式 UI 设计
- Qdrant REST API 完整封装
- TypeScript 类型定义

### 技术栈
- Vue 3 + TypeScript
- Element Plus UI 组件库
- Pinia 状态管理
- Vue Router 路由管理
- Axios HTTP 客户端
- vue-i18n 国际化

## 变更类型说明

- **新增**: 新功能
- **变更**: 现有功能的变更
- **弃用**: 即将移除的功能
- **移除**: 已移除的功能
- **修复**: Bug 修复
- **安全**: 安全相关的修复
