## [0.2.0] - 2026-02-04

### 新增
- 批量删除功能：点数据表格支持复选框多选，可批量删除选中的点
- 数据导入功能：支持从 JSON 和 CSV 文件导入点数据，包含格式验证和数据预览
- 数据导出功能：支持将点数据导出为 JSON 和 CSV 格式，提供多种导出选项：
  - 导出本页
  - 导出选中
  - 导出查询结果（集合中的所有数据）
  - 导出全部（集合中的所有数据）
- 导出格式选择对话框：使用图标按钮（JSON 和 CSV）进行格式选择
- 大数据量导出进度提示：显示加载状态和进度信息

### 变更
- 改进导出流程：选择导出范围后弹出格式选择对话框
- 优化数据加载：导出操作分批加载数据（每次 500 条），避免超时
- 优化导出性能：请求之间添加延迟，防止服务器压力过大
- 改进 UI：新建集合对话框字段更宽，避免标签文本换行

### 修复
- 修复连接状态检查：未连接时不加载集合列表
- 修复 API 错误处理：改进错误信息和连接验证
- 修复导入对话框：正确国际化文件格式提示文本

### 改进
- 改进批量操作的错误处理
- 增强数据导入导出操作的用户体验
- 改进导入导出工具函数的代码组织

## [0.1.1] - 2026-02-04

### 新增
- 集合删除功能，带警告提示对话框
- 添加点时自动生成基于毫秒时间戳的点 ID
- 向量随机生成按钮，快速创建随机向量
- Payload 随机填充功能，使用预定义结构（type, timestamp, content）
- "已有数据填充"按钮，用于加载最新点数据作为模板

### 变更
- 改进集合设置界面："已有数据填充"按钮移至底部，与取消/保存按钮对齐
- 优化集合项布局：集合名称和图标左对齐，优化操作按钮间距
- 新建集合的默认向量维度从 128 改为 1024
- 改进输入验证：向量和 Payload 验证仅在保存时进行，编辑时允许自由输入

### 修复
- 修复点 ID 格式问题：从字符串时间戳改为数字类型（无符号整数），符合 Qdrant 要求
- 修复集合设置参数加载：正确从 Qdrant 响应中读取 indexed_threshold
- 修复集合设置参数保存：正确更新 indexing_threshold 并在保存后重新加载值
- 修复 Payload 占位符文本：解决 Vue i18n 花括号解析问题
- 修复向量和 Payload 输入：移除实时验证干扰，允许用户自由输入

### 改进
- 改进集合设置更新的错误处理
- 增强集合配置的类型定义
- 改进代码组织和可维护性

## [0.1.0] - 2026-02-04

### 新增
